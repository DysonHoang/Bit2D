🌳 BIT 2D — Fenwick Tree 2D

Binary Indexed Tree mở rộng cho bài toán trên ma trận

1 Giới thiệu

BIT 2D (hay Fenwick Tree 2D) là phiên bản mở rộng của BIT 1D sang hai chiều, dùng để xử lý hiệu quả các thao tác trên ma trận:

1
Point Update — Cập nhật giá trị tại một ô (x, y)
2
Prefix Sum Query — Truy vấn tổng trên hình chữ nhật từ (1,1) đến (x,y)
3
Range Sum Query — Truy vấn tổng trên hình chữ nhật bất kỳ (r1,c1) → (r2,c2)
💡
Mỗi thao tác chỉ mất O(log N × log M) — nhanh hơn rất nhiều so với duyệt trâu O(N × M).

2 Nhắc lại BIT 1D

Trước khi hiểu 2D, hãy nắm vững 1D:

Khái niệm Lowbit

lowbit(i) = i & (-i) lấy ra bit thấp nhất được bật trong biểu diễn nhị phân của i.

i (thập phân) i (nhị phân) lowbit(i) Ý nghĩa
1 0001 1 Quản lý 1 phần tử
2 0010 2 Quản lý 2 phần tử
3 0011 1 Quản lý 1 phần tử
4 0100 4 Quản lý 4 phần tử
5 0101 1 Quản lý 1 phần tử
6 0110 2 Quản lý 2 phần tử

Hai thao tác cơ bản

C++ — BIT 1D 
// Cộng val vào vị trí i
void update(int i, int val) {
    for (; i <= n; i += i & (-i))
        bit[i] += val;
}

// Tính tổng từ 1 đến i
int query(int i) {
    int sum = 0;
    for (; i > 0; i -= i & (-i))
        sum += bit[i];
    return sum;
}
🔑
Update: nhảy lên (cộng lowbit) → ảnh hưởng tới các nút cha.
Query: nhảy xuống (trừ lowbit) → gom kết quả từ các nút con.

3 Ý tưởng BIT 2D

🧠
BIT 2D = BIT lồng BIT
Chiều thứ nhất: BIT theo hàng
Chiều thứ hai: BIT theo cột
Mỗi nút BIT theo hàng chứa một BIT theo cột bên trong.

Hình dung

Cho ma trận kích thước N × M:

c=1 c=2 c=3 c=4
r=1 a₁₁ a₁₂ a₁₃ a₁₄
r=2 a₂₁ a₂₂ a₂₃ a₂₄
r=3 a₃₁ a₃₂ a₃₃ a₃₄
r=4 a₄₁ a₄₂ a₄₃ a₄₄

query(x, y) = Tổng tất cả a[i][j] với 1 ≤ i ≤ x và 1 ≤ j ≤ y.

Cách hoạt động:

update(x, y, val) {
  🔄 Vòng ngoài: duyệt BIT theo HÀNG (i = x → N, nhảy +lowbit)
    🔄 Vòng trong: duyệt BIT theo CỘT (j = y → M, nhảy +lowbit)
      📝 bit[i][j] += val
}

4 Cài đặt chi tiết

4.1. Cấu trúc dữ liệu

Khai báo 
int bit[MAXN + 1][MAXM + 1];  // khởi tạo = 0
int N, M;                          // kích thước ma trận N hàng × M cột

4.2. Update — Cộng val vào ô (x, y)

C++ — Update 
void update(int x, int y, int val) {
    for (int i = x; i <= N; i += i & (-i))       // ← BIT theo HÀNG
        for (int j = y; j <= M; j += j & (-j))   // ← BIT theo CỘT
            bit[i][j] += val;
}
1
Bắt đầu từ hàng x, nhảy lên các hàng cha: x → x + lowbit(x) → ...
2
Tại mỗi hàng i, bắt đầu từ cột y, nhảy sang các cột cha: y → y + lowbit(y) → ...
3
Cộng val vào bit[i][j] tại mỗi ô gặp.

4.3. Query — Tổng prefix từ (1,1) đến (x,y)

C++ — Query 
int query(int x, int y) {
    int sum = 0;
    for (int i = x; i > 0; i -= i & (-i))       // ← BIT theo HÀNG (nhảy xuống)
        for (int j = y; j > 0; j -= j & (-j))   // ← BIT theo CỘT (nhảy trái)
            sum += bit[i][j];
    return sum;
}

4.4. Range Query — Tổng hình chữ nhật (r1,c1) → (r2,c2)

Dùng Nguyên lý Bao hàm - Loại trừ (Inclusion-Exclusion):

+ query(r2, c2)
Toàn bộ vùng lớn
− query(r1-1, c2)
Bỏ phần trên
− query(r2, c1-1)
Bỏ phần trái
+ query(r1-1, c1-1)
Cộng lại góc bị trừ 2 lần
1 ... c1-1 c1 ... c2
1 D D D B B B
... D D D B B B
r1-1 D D D B B B
r1 C C C ★ ★ ★
... C C C ★ ★ ★
r2 C C C ★ ★ ★

★ = Vùng cần tính  |  B, C = Bị trừ đi  |  D = Cộng lại (trừ 2 lần)

C++ — Range Query 
int queryRange(int r1, int c1, int r2, int c2) {
    return query(r2, c2) 
         - query(r1 - 1, c2) 
         - query(r2, c1 - 1) 
         + query(r1 - 1, c1 - 1);
}

5 Minh họa trực quan

Update(2, 3, 5) trên ma trận 4×4

Các ô BIT được cập nhật (tô hồng):

bit j=1 j=2 j=3 j=4
i=1 0 0 0 0
i=2 0 0 +5 +5
i=3 0 0 0 0
i=4 0 0 +5 +5
Luồng update(2, 3, 5):

i = 2 (lowbit=2):
  j = 3 → bit[2][3] += 5  |  j = 3+1 = 4 → bit[2][4] += 5  |  j = 4+4 = 8 > 4 → dừng

i = 2+2 = 4 (lowbit=4):
  j = 3 → bit[4][3] += 5  |  j = 3+1 = 4 → bit[4][4] += 5  |  j = 4+4 = 8 > 4 → dừng

i = 4+4 = 8 > 4 → dừng

Query(3, 3) — Tổng từ (1,1) đến (3,3)

bit j=1 j=2 j=3 j=4
i=1 0 0 0 0
i=2 0 đọc đọc 5
i=3 0 đọc đọc 0
i=4 0 0 5 5
Luồng query(3, 3):

i = 3 (lowbit=1):
  j = 3 → sum += bit[3][3] = 0
  j = 3-1 = 2 → sum += bit[3][2] = 0
  j = 2-2 = 0 → dừng cột

i = 3-1 = 2 (lowbit=2):
  j = 3 → sum += bit[2][3] = 5
  j = 3-1 = 2 → sum += bit[2][2] = 0
  j = 2-2 = 0 → dừng cột

i = 2-2 = 0 → dừng

✓ Kết quả: sum = 5

6 Truy vết hoàn chỉnh

Giả sử ma trận 4×4, toàn 0. Thực hiện tuần tự:

Bước 1: update(2, 3, 5)

📝 bit[2][3] += 5 → bit[2][3] = 5
📝 bit[2][4] += 5 → bit[2][4] = 5
📝 bit[4][3] += 5 → bit[4][3] = 5
📝 bit[4][4] += 5 → bit[4][4] = 5

Bước 2: update(3, 1, 7)

📝 bit[3][1] += 7 → bit[3][1] = 7
📝 bit[3][2] += 7 → bit[3][2] = 7
📝 bit[3][4] += 7 → bit[3][4] = 7
📝 bit[4][1] += 7 → bit[4][1] = 7
📝 bit[4][2] += 7 → bit[4][2] = 7
📝 bit[4][4] += 7 → bit[4][4] = 12

Trạng thái mảng BIT hiện tại:

bit j=1 j=2 j=3 j=4
i=1 0 0 0 0
i=2 0 0 5 5
i=3 7 7 0 7
i=4 7 7 5 12

Bước 3: queryRange(1, 1, 3, 3)

queryRange(1,1,3,3) = query(3,3) − query(0,3) − query(3,0) + query(0,0)

query(3,3):
  i=3: bit[3][3]=0, bit[3][2]=7 → cộng 7
  i=2: bit[2][3]=5, bit[2][2]=0 → cộng 5
  → query(3,3) = 12

query(0,3) = 0 (x=0 → vòng for không chạy)
query(3,0) = 0 (y=0 → vòng for không chạy)
query(0,0) = 0

✅ Kết quả = 12 − 0 − 0 + 0 = 12
(Đúng! Vì a[2][3]=5 và a[3][1]=7, tổng vùng (1,1)→(3,3) = 5+7 = 12)

8 Mở rộng

8.1. Update trên hình chữ nhật + Query tại điểm

Nếu cần cộng val vào toàn bộ hình chữ nhật (r1,c1)→(r2,c2) và truy vấn giá trị tại một điểm, ta dùng kỹ thuật mảng hiệu (difference array) 2D kết hợp BIT 2D:

C++ — Range Update + Point Query 
// Cộng val vào mọi ô trong HCN (r1,c1) → (r2,c2)
void rangeUpdate(int r1, int c1, int r2, int c2, int val) {
    update(r1,       c1,       +val);
    update(r1,       c2 + 1,  -val);
    update(r2 + 1,  c1,       -val);
    update(r2 + 1,  c2 + 1,  +val);
}

// Truy vấn giá trị tại điểm (x, y)
int pointQuery(int x, int y) {
    return query(x, y);   // query prefix sum = giá trị thực tại (x,y)
}
⚠️
Lưu ý: Kỹ thuật này dùng BIT để lưu mảng hiệu 2D, không phải giá trị gốc. Do đó query(x,y) cho ra giá trị thực tại ô (x,y), KHÔNG phải tổng prefix.

8.2. Range Update + Range Query

Đây là bài toán khó hơn: vừa cập nhật hình chữ nhật, vừa truy vấn tổng hình chữ nhật. Cần dùng 4 BIT 2D song song (hoặc Segment Tree 2D có Lazy).

C++ — Range Update + Range Query (4 BIT) 
long long B[4][MAXN][MAXM]; // 4 mảng BIT 2D

void update_bit(int k, int x, int y, long long val) {
    for (int i = x; i <= N; i += i & (-i))
        for (int j = y; j <= M; j += j & (-j))
            B[k][i][j] += val;
}

long long query_bit(int k, int x, int y) {
    long long s = 0;
    for (int i = x; i > 0; i -= i & (-i))
        for (int j = y; j > 0; j -= j & (-j))
            s += B[k][i][j];
    return s;
}

void rangeAdd(int x, int y, long long val) {
    update_bit(0, x, y, val);
    update_bit(1, x, y, val * x);
    update_bit(2, x, y, val * y);
    update_bit(3, x, y, val * x * y);
}

long long prefixSum(int x, int y) {
    return (x+1)*(y+1)*query_bit(0,x,y) 
         - (y+1)*query_bit(1,x,y) 
         - (x+1)*query_bit(2,x,y) 
         + query_bit(3,x,y);
}

10 Code hoàn chỉnh

C++ — Complete Solution 
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;
const int MAXM = 1005;

int bit[MAXN][MAXM];
int N, M;

void update(int x, int y, int val) {
    for (int i = x; i <= N; i += i & (-i))
        for (int j = y; j <= M; j += j & (-j))
            bit[i][j] += val;
}

int query(int x, int y) {
    int sum = 0;
    for (int i = x; i > 0; i -= i & (-i))
        for (int j = y; j > 0; j -= j & (-j))
            sum += bit[i][j];
    return sum;
}

int queryRange(int r1, int c1, int r2, int c2) {
    return query(r2, c2) 
         - query(r1 - 1, c2) 
         - query(r2, c1 - 1) 
         + query(r1 - 1, c1 - 1);
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int Q;
    cin >> N >> M >> Q;
    
    while (Q--) {
        int type;
        cin >> type;
        if (type == 1) {
            int x, y, val;
            cin >> x >> y >> val;
            update(x, y, val);
        } else {
            int r1, c1, r2, c2;
            cin >> r1 >> c1 >> r2 >> c2;
            cout << queryRange(r1, c1, r2, c2) << "\n";
        }
    }
    return 0;
}

11 Tổng kết

📋 Cheat Sheet — BIT 2D

Update
O(log N × log M)
Query
O(log N × log M)
Bộ nhớ
O(N × M)
Dòng code
~15 dòng
🔑 Ghi nhớ cốt lõi:

1. BIT 2D = Lồng 2 vòng lặp BIT 1D

2. Update(x,y,v): vòng ngoài (hàng ↑ lên), vòng trong (cột → phải)
    for i: x→N (+lowbit) | for j: y→M (+lowbit)

3. Query(x,y): vòng ngoài (hàng ↓ xuống), vòng trong (cột ← trái)
    for i: x→0 (−lowbit) | for j: y→0 (−lowbit)

4. Tổng HCN = Inclusion-Exclusion trên 4 lần gọi query
    +Q(r2,c2) −Q(r1-1,c2) −Q(r2,c1-1) +Q(r1-1,c1-1)
🎯
Mẹo thi: BIT 2D code rất ngắn, ít sai. Khi gặp bài toán trên ma trận có update điểm + query tổng hình chữ nhật, hãy nghĩ ngay đến BIT 2D trước khi xài Segment Tree 2D nặng nề hơn.
